# Entornos de Desarrollo – DEVNOVA

## Sobre mí
Estudiante de Desarrollo de Aplicaciones Multiplataforma (DAM). 
Soy un Junior Developer en formación, enfocado en aprender metodologías de trabajo profesionales, arquitecturas limpias y el ecosistema de desarrollo Java.

## Objetivo del repositorio
Este repositorio se utilizará durante el curso 2026 para aplicar y documentar las siguientes prácticas profesionales:
- **Configuración de entornos:** Setup inicial y optimización de herramientas.
- **Control de versiones:** Uso avanzado de Git y flujos de trabajo en GitHub.
- **Depuración:** Identificación y resolución de errores en tiempo de ejecución.
- **Testing:** Implementación de pruebas unitarias para asegurar la calidad del software.
- **Documentación técnica:** Elaboración de manuales y guías siguiendo estándares de industria.

## Tecnologías previstas
- **Lenguaje:** Java
- **Control de versiones:** Git / GitHub
- **IDE:** IntelliJ IDEA (Community Edition)

## Elección y Justificación del IDE
Para el desarrollo de este módulo, he seleccionado **IntelliJ IDEA** como entorno de desarrollo integrado principal.

### Justificación Técnica:
1. **Productividad:** Su motor de análisis estático ofrece sugerencias de código y refactorizaciones automáticas más precisas que las de otros IDEs tradicionales.
2. **Integración con Git:** Posee una de las mejores interfaces visuales para la gestión de ramas, resolución de conflictos y revisión de historial de commits.
3. **Estandarización:** Es el IDE de referencia en el sector profesional actual para proyectos Java, lo que facilita la transición al mercado laboral.
4. **Ecosistema:** Ofrece un soporte excelente para herramientas de construcción como Maven y Gradle, esenciales en el desarrollo moderno.
